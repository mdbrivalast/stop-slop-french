# Phrases à supprimer

## Ouvertures qui gagnent du temps

Supprimez ces phrases d'annonce. Énoncez directement le contenu.

- "Voici la chose :"
- "Voici ce que"
- "Voici ce"
- "Voici pourquoi"
- "La vérité inconfortable est"
- "Il s'avère que"
- "Le vrai problème est"
- "Soyons clairs"
- "La vérité est que"
- "Je vais le redire :"
- "Soyons honnêtes"
- "Parlons de"
- "Voici ce que j'ai trouvé intéressant"
- "Voici le problème cependant"

Toute construction "voici ce/ceci/cela" est une ouverture avant le point. Coupez-la et énoncez le point.

## Béquilles d'emphase

Ces termes n'ajoutent aucun sens. Supprimez-les.

- "Point final."
- "Laissez cela vous interpeller."
- "C'est important parce que"
- "Soyez sans doute"
- "Voici pourquoi c'est important"

## Jargon commercial

Remplacez par un langage clair.

| À éviter | Utiliser à la place |
|----------|-------------------|
| Naviguer (dans les défis) | Gérer, affronter |
| Dépacker (analyse) | Expliquer, examiner |
| S'engager dedans | Accepter, embrasser |
| Panorama (contexte) | Situation, domaine |
| Révolutionnaire | Significatif, important |
| Doubler la mise | S'engager, augmenter |
| Approfondissement | Analyse, examen |
| Prendre du recul | Reconsidérer |
| Aller de l'avant | Ensuite, dorénavant |
| Revenir sur | Retourner à, revisiter |
| Sur la même longueur d'onde | Aligné, d'accord |

## Adverbes

Éliminez tous les adverbes. Pas de mots en -ment. Pas d'adoucisseurs, pas d'intensificateurs, pas d'atténuateurs.

Cibles spécifiques :

- "vraiment"
- "juste"
- "littéralement"
- "sincèrement"
- "honnêtement"
- "simplement"
- "en réalité"
- "profondément"
- "réellement"
- "fondamentalement"
- "intrinsèquement"
- "inévitablement"
- "intéressamment"
- "importante"
- "cruciale"
- "absolument"
- "particulièrement"
- "essentiellement"

Coupez aussi ces phrases de remplissage :

- "En réalité"
- "De nos jours"
- "Il vaut la peine de noter"
- "En fin de compte"
- "Quand il s'agit de"
- "Dans un monde où"
- "La réalité est"
- "Au cœur de"

## Méta-commentaires

Supprimez les digressions autoréférencées. L'essai devrait avancer, pas annoncer sa propre structure.

- "Indice :"
- "Coup de théâtre :" / "Spoiler :"
- "Vous savez déjà cela, mais"
- "Mais c'est un autre article"
- "X est une caractéristique, pas un bug"
- "Déguisé en"
- "Le reste de cet essai explique..."
- "Laissez-moi vous expliquer..."
- "Dans cette section, nous..."
- "Comme nous le verrons..."
- "Je veux explorer..."

## Emphase performative

Fausse intimité ou sincérité manufacturée :

- "s'insinue"
- "Je vous le promets"
- "Ils existent, je vous le promets"

## Montrer au lieu de dire

Annoncer la difficulté ou l'importance plutôt que la démontrer :

- "C'est vraiment difficile"
- "C'est ça le vrai leadership"
- "C'est ça que X ressemble vraiment"
- "compte vraiment"

## Déclaratives vagues

Des phrases qui annoncent de l'importance sans nommer la chose spécifique. Éliminez-les.

- "Les raisons sont structurelles"
- "Les implications sont significatives"
- "C'est le problème le plus profond"
- "Les enjeux sont importants"
- "Les conséquences sont réelles"

Si une phrase dit que quelque chose est important/profond/structurel sans montrer la chose spécifique, coupez-la ou remplacez-la par la chose spécifique.
